# Seasonal Hospitality Performance Dashboard

## Project Overview

This project focuses on analyzing seasonal trends in the hospitality industry using Power BI. The dashboard provides insights into booking patterns, revenue performance, occupancy rates, and customer behavior across different seasons.

The goal of the project is to help hospitality businesses understand how seasonal demand impacts revenue and occupancy so they can optimize pricing strategies, staffing, and marketing campaigns.

Power BI was used to transform raw hospitality data into an interactive dashboard that enables quick and data-driven decision making.

## Business Problem

The hospitality industry experiences strong seasonal fluctuations in demand. Hotels and resorts must understand these patterns to manage pricing, staffing, and marketing effectively.

Key questions addressed in this project:

• Which seasons generate the highest revenue?  
• How do booking patterns change throughout the year?  
• What are the peak occupancy periods?  
• Which locations or room types perform best during high season?  
• How can hotels optimize their operations based on seasonal demand?

Understanding these insights helps hospitality businesses improve revenue management and operational efficiency.

## Dashboard Features

The Power BI dashboard includes several interactive visualizations:

### Revenue Analysis
Displays total revenue generated across different seasons and months.

### Occupancy Rate Analysis
Shows room occupancy patterns during peak and off seasons.

### Booking Trends
Analyzes booking volume trends over time.

### Property Performance
Compares performance across different hotel properties.

### Seasonal Demand Patterns
Highlights the impact of seasonal tourism trends on hospitality performance.

Interactive slicers allow filtering by:

• Month  
• Season  
• Property  
• Room category

## Key Insights

• Peak revenue occurs during high tourism seasons.  
• Booking demand increases significantly during holiday periods.  
• Certain room categories generate higher revenue during peak months.  
• Off-season periods show lower occupancy but present opportunities for promotional strategies.

These insights help hospitality businesses plan pricing strategies, marketing campaigns, and resource allocation.
